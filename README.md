# 🚀 Netflix CI/CD Pipeline Project

## 📌 Project Overview

This project demonstrates a complete **CI/CD pipeline setup** using DevOps tools to automate build, test, and deployment processes.

The pipeline integrates source code management, continuous integration, and containerized deployment.

---

## 🛠 Tech Stack

* Jenkins
* Docker
* AWS (EC2 / S3 / IAM)
* Git & GitHub
* Linux

---

## ⚙️ Key Features

* Automated CI/CD pipeline using Jenkins
* Docker containerization for application deployment
* Integration with GitHub for source code management
* Continuous deployment on AWS EC2
* Reduced manual deployment effort

---

## 🏗 Architecture

1. Developer pushes code to GitHub
2. Jenkins triggers build automatically
3. Docker image is created
4. Application is deployed on AWS EC2
